Requirements

Server/System Requirements

Before installing this script, ensure that your server meets the following requirements:

Requirements information
  • PHP Version 7.3 or above
  • MySQL Version 5.7+ or MariaDB version 10.2+
  • Ctype PHP Extension
  • Fileinfo PHP extension
  • JSON PHP Extension
  • Mbstring PHP Extension
  • OpenSSL PHP Extension
  • PDO PHP Extension
  • PDO_MYSQL PHP Extension
  • Tokenizer PHP Extension
  • XML PHP Extension
  • CURL PHP Extension
  • GD PHP Extension
  • Folder Permission 775/777

These requirements are essential to guarantee a smooth installation and optimal performance of the script. Make sure your server environment aligns with these specifications before proceeding with the installation process.

Installation Process

Offline Installation Guide for PrimeStock

Follow these steps to successfully install your PrimeStock on your localhost PC:

Step 1: Extract ZIP and Place Files
  1. Extract the main installation files from the downloaded archive.
  2. Then go to script folder and extract prime_stock.zip. After that copy this file and paste it in www directory (for WAMP) or htdocs directory (for XAMPP).
Step 2: Create a Database
  1. Open your web browser and navigate to phpMyAdmin.
  2. Create a new database that will be used by application.
Step 3: Access the Installation Wizard
  1. Launch your web browser.
  2. Enter the following URL in the address bar: http://localhost/myscript (replace myscript with the appropriate folder name you used in Step 2).
Step 4: Installation Wizard
  1. The installation wizard will load, Now Enter Database Details
  2. The wizard check all extension and PHP version are suitable or not
  3. Also this step check all permission and ready to install
  4. Follow the steps in the installation wizard until you reach the final step.
Step 5: Installation Success
  1. Once the installation is successful, you will see a message confirming the installation.
  2. The message will also provide you with default access credentials for logging in. 
                                                        For Admin Default Aceess
                                                        Username: admin@gmail.com
                                                        Password: password

Please note that these steps are a general guideline. Depending on your local server configuration and any specific instructions provided with the script, there might be slight variations in the installation process. Always refer to the script's documentation or readme file for any additional information.

Installation Process

Installation Guide for Prime Stock on cPanel

Follow these steps to successfully install your Prime Stock on your cPanel hosting:

Step 1: Extract and Upload Files
  1. Extract the main installation files from the downloaded archive.
  2. After that you can find two folder script and Documents. Go to script folder and Upload this prime_stock.zip file in you cpanel file manager.
Step 2: Create a Database
  1. Log in to your cPanel account.
  2. Navigate to the MySQL Databases section and create a new database for your Laravel application.
Step 3: Configure Database
  1. While still in cPanel, go to the MySQL Databases section and create a new database user.
  2. Assign the user to the database you created and grant necessary privileges.
Step 4: Access the Installation Wizard
  1. Launch your web browser.
  2. Enter the following URL in the address bar: http://yourdomain.com or http://yourdomain.com/myscript (replace myscript with the appropriate folder name you used in Step 1).
Step 5: Installation Wizard
  1. The installation wizard will load, Now Enter Database Details
  2. This Wizard Will checking if your server meets the requirements. Make sure your server meets all the requirements listed.
  3. The wizard will also check for necessary file permissions. If any file or directory requires permissions changes, follow the instructions provided.
  4. Follow the steps in the installation wizard until you reach the final step.
Step 6: Installation Success
  1. Once the installation is successful, you will see a message confirming the installation.
  2. The message will also provide you with default access credentials for logging in. 
                                                        For Admin Default Aceess
                                                        Username: admin@gmail.com
                                                        Password: password

Please keep in mind that the steps provided are general guidelines. Depending on your hosting provider and any specific instructions from the script's documentation, the installation process might have variations. Always refer to the script's documentation or readme for any specific guidance.
